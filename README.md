# 💪 Personal Trainer Web

Proyecto web desarrollado como práctica de desarrollo frontend. Simula el sitio web de un personal trainer con información sobre servicios, galería, presupuestos y contacto.

---

## 🌐 Demo en vivo

🔗 [https://alipidev.github.io/webPersonalTrainer/]

---

## 📄 Páginas

| Página | Descripción |
|---|---|
| `index.html` | Inicio con servicios, noticias y opiniones |
| `views/galeria.html` | Galería de imágenes |
| `views/presupuesto.html` | Formulario de presupuesto |
| `views/contacto.html` | Formulario de contacto |
| `views/aviso-legal.html` | Aviso legal |

---

## 🛠️ Tecnologías utilizadas

- **HTML5** — Estructura semántica
- **CSS3** — Estilos y diseño responsive
- **JavaScript (Vanilla)** — Carga dinámica de componentes, menú hamburguesa y link activo
- **JSON** — Carga dinámica de noticias
- **Font Awesome** — Iconos de redes sociales
- **GitHub Pages** — Despliegue

---

## ✨ Funcionalidades

- ✅ Menú de navegación responsive con botón hamburguesa
- ✅ Link activo resaltado automáticamente según la página actual
- ✅ Noticias cargadas desde un archivo JSON externo
- ✅ Compatible con GitHub Pages (rutas dinámicas)
